Full-stack build — running

One backend, two doors in for the farmer.

This is the working version of BAO's inclusivity system: a single Node server holds live prices, buyers and bookings in SQLite, and pushes updates out over Server-Sent Events the instant a trader changes a number. A USSD adapter and a WhatsApp adapter both sit on top of the exact same conversation engine, so a farmer gets the same prices and the same buyers no matter which door they walk through.
